# MyElephant-ClockEST 🐘🕒

A lightweight Eastern Standard Time (EST) clock generated with clean HTML and CSS. This tool is designed to be easily embedded into any web project or dashboard.

## 🛠 Features
- **Real-time EST:** Always accurate to Eastern Standard Time.
- **Minimalist Design:** Styled with CSS for a modern look.
- **Easy Integration:** Just drop the files into your web server.

## 🚀 How to Use
1. **Clone the Repo:** `git clone https://github.com`
2. **Open the Clock:** Simply open `clock.html` in any web browser.
